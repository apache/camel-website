We are pleased to announce the 3.20.4 LTS release of Camel Quarkus.

Camel Quarkus 3.20.4 aligns with the Camel 4.10.8 LTS and Quarkus 3.20.5 LTS releases.

For more information, please check the [Release notes](/releases/q-3.20.4/).

## Camel 4.10.8

Please refer to the [Camel 4.10.8 release announcement](/blog/2025/12/RELEASE-4.10.8/) for more details.

## Quarkus 3.20.5

Quarkus 3.20.5 is currently scheduled for release on January 21st. The quarkus-camel-bom in this release will be aligned with Camel Quarkus 3.20.4.

Keep a watch on the Quarkus release announcements for more details.
