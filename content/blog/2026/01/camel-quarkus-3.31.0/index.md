We are pleased to announce the 3.31.0 release of Apache Camel Quarkus.

Camel Quarkus 3.31.0 aligns with the Camel 4.17.0 and Quarkus 3.31.0 releases.

For more information, please check the [Release notes](/releases/q-3.31.0/).

## Camel 4.17.0

Please refer to the [Camel 4.17.0 release announcement](/blog/2026/01/RELEASE-4.17.0/) and the [Camel 4.17 what's new blog post](/blog/2026/01/camel417-whatsnew/) for more details.

## Quarkus 3.31.0

Please refer to the [Quarkus 3.31.0 release notes](https://github.com/quarkusio/quarkus/releases/tag/3.31.0) for more details.
You can find the full release announcement on the [Quarkus Blog](https://quarkus.io/blog/tag/release/).

## Migration guide

Refer to the [migration guide](/camel-quarkus/next/migration-guide/3.31.0.html) for details of noteworthy changes that were made since the 3.30.0 release.
