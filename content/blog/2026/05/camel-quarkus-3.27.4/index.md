We are pleased to announce the 3.27.4 LTS release of Camel Quarkus.

Camel Quarkus 3.27.4 aligns with the Camel 4.14.7 LTS and Quarkus 3.27.3.1 LTS releases.

For more information, please check the [Release notes](/releases/q-3.27.4/).

## Camel 4.14.7

Please refer to the [Camel 4.14.7 release announcement](/blog/2026/04/RELEASE-4.14.7/) for more details.

## Quarkus 3.27.3.1

You can find the Quarkus 3.27.3.1 release announcement on the [Quarkus Blog](https://quarkus.io/blog/CVE-2026-39852/).

## Quarkus 3.27.4

Quarkus 3.27.4 is currently scheduled for release on May 26th. The `quarkus-camel-bom` in this release will be aligned with Camel Quarkus 3.27.4.

Keep a watch on the Quarkus release announcements for more details.
